COMPLETE
  • JWT Bearer tokens (30-day, HS256) replacing header-based auth
  • signToken/verifyToken utilities, verifyAuth middleware
  • Token stored in AsyncStorage, attached via apiFetch()
  • Parent PIN guard: token held until PIN verified

2. Global Auth Guard

COMPLETE
  • All /api/* routes require verifyAuth by default
