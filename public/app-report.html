🔔 Push Notification Triggers (9 active)

📋 Attendance Alert
POST /api/attendance/save
Fires for each absent student → parent. Reads parentId from students collection.
🚌 Bus Approaching
checkProximityAlerts()
Fires when bus is within 500m of stop → parent. Looks up parentId via Admin SDK.
✅ Boarded Bus
POST /api/trip/scan (board)
Fires on first QR scan of the day (boarding) → parent. Uses studentData.parentId.
🏠 Safely Home
POST /api/trip/scan (deboard)
Fires on second QR scan of the day (deboard) → parent.
✅/❌ Student Leave
POST /api/leave-request/update-status
Fires on Approved or Rejected → parent. Uses leaveData.parentId.
✅/❌ Staff Leave
POST /api/leave-request/update-status
Fires on Approved or Rejected → teacher/driver/cleaner. Uses leaveData.staffId.
🔔 Fee Reminder
POST /api/admin/fees/send-reminder
Fires per student → parent. Looks up parentId from students collection per sid.
📁 New Document
POST /api/student-files (upload)
Fires after file uploaded → parent. Looks up parentId from students collection.
📝 Marks Updated
POST /api/marks/save
Fires per student after marks saved → parent. Reads parentId from existing studentSnap.

Token registration: POST /api/notifications/register-token · Tokens stored in Firestore fcm_tokens collection · Push via Expo Push API (exp.host/--/api/v2/push/send)
