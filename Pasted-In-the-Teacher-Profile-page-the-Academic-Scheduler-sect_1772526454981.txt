1. Fix the "Assign Class" Dropdown — Populate with Real Classes

Do not hardcode class names — fetch them dynamically from the classes table.
2. Class Teacher Assignment Rules
Only one teacher can be assigned as Class Teacher of a given class at a time.
If a class already has a Class Teacher assigned, prevent reassigning it to another teacher and show a clear warning message.

Class (dropdown — all available classes)
Subject (dropdown or text)
Day (Monday to Saturday)
Start Time (time picker)
End Time (time picker)

After saving, all assigned class slots must be listed clearly in the Academic Scheduler section showing: Class name, Subject, Day, Start Time – End Time.
The count "0 classes" must update to reflect the real number of assigned slots.
